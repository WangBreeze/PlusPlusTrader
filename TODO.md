## 依赖管理

### 必需依赖：
- CMake 3.10+
- C++17编译器
- Python 3.6+（可选）

### 可选依赖：
- Boost（高级功能）
- pybind11（Python绑定）
- SQLite3（数据存储）
- curl（HTTP客户端）

## 构建和安装

```bash
# 编译
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j4

# 运行示例
./example

# 安装（后续）
sudo make install
```
