2. **配置认证**:
   ```bash

   ```

### 备选方案：添加SSH密钥
1. **查看SSH公钥**:
   ```bash
   cat ~/.ssh/id_ed25519.pub
   ```

2. **添加到GitHub**:
   - 访问 https://github.com/settings/keys
   - 点击 "New SSH key"
   - 粘贴公钥内容并保存

3. **测试并推送**:
   ```bash
   ssh -T git@github.com
   git remote set-url origin git@github.com:WangBreeze/PlusPlusTrader.git
   git push origin master
   ```
