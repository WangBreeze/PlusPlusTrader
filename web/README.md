# PlusPlusTrader Web界面

## 🎯 概述

PlusPlusTrader Web界面提供了一个直观的图形化界面，用于监控和管理量化交易系统。界面包含四个主要功能模块：

1. **📊 仪表盘** - 实时数据监控和系统状态
2. **📚 使用指南** - 完整的系统文档和使用说明
3. **💡 示例代码** - 实用的Python代码示例
4. **⚙️ 系统配置** - 配置文件和参数说明

## 🚀 快速开始

### 方法1：使用简化版（推荐）
简化版不依赖外部Python包，开箱即用：

```bash
cd /home/wanglc/.openclaw/workspace/PlusPlusTrader/web
./start_web.sh
```

### 方法2：使用完整版（需要安装依赖）
完整版使用Dash框架，提供更丰富的交互功能：

```bash
cd /home/wanglc/.openclaw/workspace/PlusPlusTrader/web

# 安装依赖（如果尚未安装）
pip install dash dash-bootstrap-components plotly pandas flask

# 启动应用
python3 app.py
```

## 🌐 访问界面

启动后，在浏览器中访问：
- **简化版**: http://127.0.0.1:8050 (端口可能自动调整)
- **完整版**: http://127.0.0.1:8050

## 🔧 技术架构

### 简化版
- **框架**: 纯Python + 标准HTTP服务器
- **前端**: 纯HTML/CSS/JavaScript
- **依赖**: 无外部依赖
- **特点**: 轻量级、快速启动、易于部署

### 完整版
- **后端**: Dash (基于Flask) + Plotly
- **前端**: React + Bootstrap组件
- **依赖**: 需要安装多个Python包
- **特点**: 丰富的交互功能、实时图表、更好的用户体验

## 📊 数据源

Web界面支持多种数据源：
- **模拟数据**: 内置的随机数据生成器
- **币安(Binance)**: 实时市场数据（需要API密钥）
- **欧易(OKX)**: 实时市场数据（需要API密钥）

## ⚡ 性能特性

- **实时更新**: 数据自动刷新，保持最新状态
- **响应式设计**: 适配各种屏幕尺寸
- **轻量级**: 简化版占用资源极少
- **易于扩展**: 模块化设计，便于添加新功能

## 🐛 故障排除

### 常见问题

1. **端口被占用**
   ```
   ./start_web.sh 会自动检测并选择可用端口
   ```

2. **Python依赖问题**
   ```
   # 使用简化版避免依赖问题
   ./start_web.sh
   ```

3. **无法访问界面**
   ```
   # 检查防火墙设置
   sudo ufw allow 8050/tcp
   ```

4. **数据不更新**
   ```
   # 点击"启动数据流"按钮
   # 检查网络连接
   ```
