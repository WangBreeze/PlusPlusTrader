### 已实现的指标：
1. **移动平均线 (MA)**
   - MA5: 5日移动平均线
   - MA10: 10日移动平均线  
   - MA20: 20日移动平均线
   - MA60: 60日移动平均线

2. **相对强弱指数 (RSI)**
   - 周期: 14日
   - 范围: 0-100
   - 超买线: 70
   - 超卖线: 30

3. **MACD指标**
   - 快线周期: 12日
   - 慢线周期: 26日
   - 信号线: 9日

### 指标计算算法：
```python
# 移动平均线计算
def calculate_ma(data, period):
    return sum(item['close'] for item in data[-period:]) / period

# RSI计算
def calculate_rsi(data, period=14):
    gains = sum(max(0, data[i]['close'] - data[i-1]['close']) 
                for i in range(1, len(data)))
    losses = sum(max(0, data[i-1]['close'] - data[i]['close']) 
                 for i in range(1, len(data)))
    rs = gains / losses if losses > 0 else float('inf')
    return 100 - (100 / (1 + rs))
```
